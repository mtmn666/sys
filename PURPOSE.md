# プロジェクトの目的（Purpose）

ドル円（USD/JPY）の金利差モデルを使った自動リサーチAIを構築する。

## 入力

| フィールド | 説明 |
|---|---|
| `us10y` | 米10年債利回り |
| `jp10y` | 日本10年債利回り |
| `spread_yesterday` | 前日の金利差 |
| `usd_jpy` | ドル円レート |
| `news_list` | 今日のニュースリスト |

## 出力

- 金利差の分析（今日の spread 計算）
- 前日比の変化（spread_change）
- ドル円への一般的な影響（drivers）
- 主要材料トップ3（top_news）
- センチメント（強気 / 弱気 / 中立）
- 重要度スコア（0〜100）

## 実行環境

| ツール | 役割 |
|---|---|
| n8n | 自動実行・スケジューリング |
| Claude API | 分析エンジン |
| VSCode | コード管理 |
| GitHub | バージョン管理 |
| ngrok | 外部アクセス・webhook受信 |

## 拡張方針（A → B → C）

- **A（現在）**: 金利差モデルのプロトタイプ。手動データ入力 or 固定値で動作確認。
- **B（次）**: 外部APIからのリアルタイムデータ取得（US10Y・JP10Y・USD/JPY）を自動化。
- **C（将来）**: VIX・株価・2年債などの追加変数、Notion/Slack への自動投稿。
